Import Advanced Options

Another collapsible panel. The following variables can be set expandable panels:

  1. panelspeed: defines panel expand/collapse speed in milliseconds
  2. totalpanels: the total number of colapsible panels used
  3. defaultopenpanel: leave at 0 to keep all panels initially closed
  4. accordian: sets accordion effect (only one panel can remain open at once)
